API Documentation

Authentication

  • POST /api/auth/register - Register new client
  • POST /api/auth/login - Authenticate client
  • POST /api/auth/refresh - Refresh authentication token

Location Data

  • POST /api/locations - Send GPS coordinates
  • POST /api/locations/batch - Send multiple coordinates
  • GET /api/locations/{client_id} - Get client locations
  • GET /api/locations/{client_id}/latest - Get latest location

Client Management

  • GET /api/clients - List all clients
  • GET /api/clients/{client_id} - Get client details
  • PUT /api/clients/{client_id} - Update client
  • DELETE /api/clients/{client_id} - Deactivate client

Visualization

  • GET /api/routes/{client_id} - Get route data
  • GET /web/map - Web map interface
  • GET /api/logs/{client_id} - Get client logs
